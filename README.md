# ESP32-Mini-Arcade-Flappy-Bird-Dino-Game
Turn your ESP32 into a pocket-sized retro arcade! Play Flappy Bird and the Chrome Dino game on a tiny OLED screen using a single touch sensor. Features a custom physics engine, memory-efficient PROGMEM pixel-art sprites, and seamless one-tap controls. A perfect, minimalist DIY hardware project combining classic 8-bit fun with embedded C++ design.
